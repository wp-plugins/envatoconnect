== Description ==

EnvatoConnect plugin lets Envato marketplace’s users to connect (ie login) with your WordPress site using their valid account username and API key.

[Visit the Plugin's Official Page](http://envatoconnect.wizylabs.com/)

== Installation ==

Upload the plugin's .zip file to your WordPress site using the WordPress plugins uploader (WordPress Admin > Plugins > Add New > Upload). Activate the plugin and you are done!
